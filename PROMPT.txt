================================================================================
E-COMMERCE WEBSITE MVP - COMPLETE DEVELOPMENT PROMPT
================================================================================

ROLE:
You are a Full Stack Developer. Build a complete MVP for an E-Commerce Website 
using Django for the backend and React for the frontend.

================================================================================
TASK:
================================================================================

Create a minimal viable e-commerce website that enables users to:
- Browse and explore products
- View individual product details
- Add and remove products from a shopping cart
- Handle basic product CRUD operations (admin access)
- Search and filter products
- User authentication (login/register/logout)
- Wishlist functionality
- Product reviews and ratings

Focus on: Clean UI, modular architecture, and smooth integration between 
frontend and backend.

================================================================================
TECH STACK:
================================================================================

FRONTEND:
- Framework: React.js 18.2.0
- Routing: React Router DOM 6.11.0
- State Management: React Context API
- HTTP Client: Axios 1.4.0
- Styling: CSS3 (Custom responsive design)

BACKEND:
- Framework: Django 5.2.8
- API Framework: Django REST Framework 3.16.1
- Database: SQLite (for MVP)
- Authentication: Token-based (Django REST Framework)
- CORS: Django CORS Headers 4.9.0

API COMMUNICATION:
- REST API with JSON
- Token authentication
- CORS enabled for cross-origin requests
