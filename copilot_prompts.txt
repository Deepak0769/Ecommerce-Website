# Copilot Prompts Used for E-Commerce MVP

## Backend Development Prompts

1. "Create Django models for Product with fields: name, description, price, image_url, stock, created_at, updated_at"

2. "Create Django model for CartItem with session_id, product foreign key, quantity, and computed total_price property"

3. "Implement ProductSerializer using Django REST Framework with all product fields"

4. "Create CartItemSerializer with nested ProductSerializer and computed total_price field"

5. "Implement ProductViewSet with ModelViewSet for full CRUD operations"

6. "Create CartViewSet with custom actions for add, remove, and clear cart operations"

7. "Configure Django settings.py with REST Framework, CORS headers, and SQLite database"

8. "Set up URL routing for /api/products/ and /api/cart/ endpoints"

9. "Create Django admin configuration for Product and CartItem models"

10. "Configure CORS to allow requests from http://localhost:3000"
